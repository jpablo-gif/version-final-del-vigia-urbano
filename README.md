# 🚨 VigíaUrbano

**Sistema de alertas tempranas para seguridad ciudadana**

---

## 📋 Descripción

VigíaUrbano es una aplicación de escritorio desarrollada en Java que permite a los ciudadanos reportar incidentes de seguridad en tiempo real. El sistema clasifica automáticamente los reportes y notifica a las autoridades correspondientes según el tipo de incidente.

---

## 🎯 Funcionalidades

### Actuales (MVP)
- ✅ Gestión de usuarios (Civiles y Autoridades)
- ✅ Reporte de incidentes (Tiroteo, Incendio, Robo, Sospechoso, Otro)
- ✅ Clasificación automática de organismos responsables
- ✅ Persistencia en archivos CSV
- ✅ Reportes tabulados tipo Excel

### Próximas (En desarrollo)
- ⏳ Interfaz gráfica con Swing
- ⏳ Diálogo de reporte con advertencia de 10 segundos
- ⏳ Estadísticas por categoría y ubicación
- ⏳ Login con roles diferenciados
