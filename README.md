# 🚨 VigíaUrbano

**Sistema de alertas tempranas para seguridad ciudadana**

---

## 📋 Descripción

VigíaUrbano es una aplicación de escritorio desarrollada en Java que permite a los ciudadanos reportar incidentes de seguridad en tiempo real. El sistema clasifica automáticamente los reportes y notifica a las autoridades correspondientes según el tipo de incidente.
